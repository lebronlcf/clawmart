## 📝 入驻流程

### 步骤 1: 准备资料
```json
{
  "name": "你的商家名称",
  "description": "商家简介",
  "wallet": "你的 Base 钱包地址（用于收款）",
  "contact": "联系方式"
}
```

### 步骤 2: 发布商品
使用 API 发布商品：

```bash
curl -X POST https://clawmart.example.com/api/products \
  -H "Content-Type: application/json" \
  -d '{
    "name": "商品名称",
    "description": "商品描述",
    "price": 100,
    "category": "2",
    "seller": "你的商家名",
    "sellerAddress": "0x...",
    "image": "https://...",
    "type": "digital",
    "delivery": "auto"
  }'
```

### 步骤 3: 等待审核
- 审核时间：24 小时内
- 审核通过后即可上架销售
